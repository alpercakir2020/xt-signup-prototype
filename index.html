Implementation notes for Vijay

  • → Font: Inter (400/500/600/700/800 weights) — switching from Proxima Nova
  • → Primary CTA: #ee7271 — matches existing .btn color in app CSS
  • → Inputs: white bg #ffffff, border 1px solid #d1d5db, height 52px, radius 6px
  • → Page layout: white .page + .page-inner (max 520px, no card shadow)
  • → URL field: .slug-row flex — .slug-input (rounded left) + .slug-suffix (gray, rounded right)
  • → Feature grid on unlock screen: .feature-grid 2-col CSS grid, tiles use .feature-item
  • → Progress dots: .progress-wrap + .step-dot.done/.active/.upcoming
  • → All design tokens in styles.css — CSS variable names match app conventions
